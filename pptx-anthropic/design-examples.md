# PPT 精進設計範例

這個範例展示如何應用 Professional Design Principles 創建高品質簡報。

## 範例：OpenClaw 教學簡報

### 設計系統設定

```python
from pptx import Presentation
from pptx.util import Inches, Pt
from pptx.enum.text import PP_ALIGN
from pptx.enum.shapes import MSO_SHAPE
from pptx.dml.color import RGBColor

# === 設計系統 ===
# 字體層次（大對比）
H1_SIZE = Pt(52)      # 主標題
H2_SIZE = Pt(36)      # 頁標題  
H3_SIZE = Pt(28)      # 副標題
BODY_SIZE = Pt(22)    # 內文
CAPTION_SIZE = Pt(16) # 輔助文字

# 間距系統（留白）
MARGIN = Inches(0.6)  # 邊距
SPACING_L = Pt(28)    # 大行距
SPACING_M = Pt(16)    # 中等行距

# 顏色系統
PRIMARY = (37, 99, 235)      # 藍-600
DARK = (15, 23, 42)          # Slate-900
GRAY = (71, 85, 105)         # Slate-600
WHITE = (255, 255, 255)

# 跨平台中文字體
FONT_TC = "Microsoft JhengHei"
```

### 顏色應用範例

```python
# Slate 系統（推薦）

```

### 字體設定範例

```python
def set_font(run, size, bold=False, color=DARK):
    """統一字體設定"""
    run.font.name = "Microsoft JhengHei"  # 跨平台中文
    run.font.size = size
    run.font.bold = bold
    run.font.color.rgb = RGBColor(*color)

# 使用範例
set_font(p.runs[0], H1_SIZE, True, WHITE)   # 主標題
set_font(p.runs[0], H2_SIZE, True, DARK)    # 頁標題
set_font(p.runs[0], BODY_SIZE, False, GRAY) # 內文
```

### 檢查清單

創建完成後檢查：

- [ ] 字體層次清晰（標題:內文 ≥ 2:1）
- [ ] 留白充足（邊距 ≥ 0.5"）
- [ ] 6×6 法則（每頁 ≤ 6 行）
- [ ] 版面多樣化（不重複同一版面）
- [ ] 對齊一致（主要內容左對齊）
- [ ] 對比度足夠（文字清晰可讀）
- [ ] 顏色統一（使用設計系統）
- [ ] 跨平台字體（Microsoft JhengHei）
